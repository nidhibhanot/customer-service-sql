# Customer Service SQL Analytics

This repository demonstrates **SQL skills applied to customer service operations**. It includes sample data, SQL queries, and analysis for:

- Total calls per agent
- Average call duration (AHT)
- Customer satisfaction (CSAT) analysis
- Issue type tracking
- Escalation handling
- Repeat customer detection

## Dataset

`customer_service_data.csv` contains:

- CallID: Unique ID per call  
- AgentID: Employee handling the call  
- CustomerID: Unique customer identifier  
- CallDate: Date of call  
- CallDurationMinutes: Duration in minutes  
- IssueType: Type of customer issue (Billing, Technical, Complaint, General)  
- ResolutionStatus: Resolved / Escalated  
- CSATScore: Customer satisfaction rating (1â€“5)

`queries.sql` includes practical queries for:

- Agent performance metrics  
- Customer satisfaction analysis  
- Identifying escalated or long calls  
- Detecting repeat customers
